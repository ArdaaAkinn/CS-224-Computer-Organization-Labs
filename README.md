# CS-224-Computer-Organization-Labs

This repository contains laboratory assignments completed for the CS-224 Computer Organization course.

The projects focus on computer architecture, processor design, MIPS architecture, low-level programming, and hardware implementation concepts using Assembly, SystemVerilog, and Vivado tools.

## Technologies Used

- MIPS Assembly
- SystemVerilog
- Vivado Design Suite
- FPGA Development Tools

### LAB 1 — MIPS Assembly Language Programming

Introduction to MIPS assembly programming fundamentals.

Topics covered:
- MIPS instruction set
- Register operations
- Arithmetic and logical instructions
- Branching and control flow
- Basic memory operations

### LAB 2 — MIPS Assembly Language Programming: Linked Lists, Recursion, Program as Data

Advanced MIPS assembly programming concepts.

Topics covered:
- Linked list implementations
- Recursive procedures
- Stack management
- Function calls
- Treating programs as data

### LAB 3 — MIPS Single-Cycle Datapath and Controller

Implementation and analysis of a single-cycle MIPS processor datapath and control unit.

Topics covered:
- Single-cycle processor architecture
- Datapath components
- Control signal generation
- ALU operations
- Instruction execution flow

### LAB 4 — Pipelined Microarchitecture

Implementation of pipelined processor architecture concepts.

Topics covered:
- Pipeline stages
- Instruction pipelining
- Hazard detection
- Data forwarding
- Pipeline performance concepts

### LAB 5 — Cache Memory Principles & Practicing Memory Mapped I/O

Exploration of cache organization and memory-mapped I/O principles.

Topics covered:
- Cache memory fundamentals
- Address mapping
- Memory hierarchy
- Memory-mapped I/O
- Hardware-level memory interaction
